Preperation:

1- Change the headers at line 8 on GE Update Price.

2- Change price history directory (you may need to create the folders). Data takes aroun 550 MB but you may reduce this by editing the earliest day value in GE Update Price (default is 90 days).


Usage:

1- Run GE Update Price, wait for it to finish.

2- Run GEWatch.

3- The analysis results will be in the Results folder.

4- Buy items with high CBPR and sell items with high CSPR. Try to trade items with similiar CBPR accross different days (CBPR1d, CBPR3d, CBPR7d)
